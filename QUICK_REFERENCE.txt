HOW IT WORKS
================================================================================

ADMIN WORKFLOW:
  1. Configure tier thresholds (S/A/B/C/D for each event)
  2. Configure grade multipliers (adjust fairness)
  3. Enable/disable badges
  4. During competition: Enter student times
  5. Click "Complete Round"
  6. System calculates points automatically
  7. Results appear in all leaderboards instantly

STUDENT WORKFLOW:
  1. Compete in event (admin enters their times)
  2. Check profile: See points earned
  3. Check school page: See team standing
  4. Check achievements: See badges earned
  5. Understand multiplier: Grade 7 gets 1.70x, Grade 12 gets 1.0x

SPECTATOR WORKFLOW:
  1. Go to projector display board
  2. Watch live leaderboard update every 3 seconds
  3. See schools or individuals ranking
  4. Understand school divisions (A/B/C by size)

POINT CALCULATION EXAMPLE
================================================================================

Student: Sarah Chen | Grade 7 | 3x3 Cube
Solve Times: 35.23, 34.89, 35.12, 33.45, 34.67 sec

Step 1: Determine Tier
  - Best: 33.45s → A tier (20-45s range)
  - Avg: 34.67s → A tier
  - Base points: 5 pts each

Step 2: Apply Grade Multiplier
  - Grade 7 multiplier: 1.70x
  - Best time: 5 × 1.70 = 8.5 pts
  - Avg time: 5 × 1.70 = 8.5 pts

Step 3: Check Bonuses
  - PB? Yes! (beat previous) → +1 × 1.70 = 1.7 pts
  - Clutch? Yes! (PB in Finals) → +2 × 1.70 = 3.4 pts
  - Streak? No (only 1 improvement)
  - School Momentum? Yes! (school had 0 DNFs) → +5 × 1.70 = 8.5 pts

Step 4: Total
  8.5 + 8.5 + 1.7 + 3.4 + 8.5 = 30.6 POINTS

For School:
  Sum of all 8 students' points = ~216 total school points

KEY CONCEPTS
================================================================================

TIER SYSTEM:
  S tier: <20 sec → 10 points (excellent)
  A tier: 20-45 sec → 5 points (great)
  B tier: 45-60 sec → 2 points (good)
  C tier: 60-120 sec → 1 point (average)
  D tier: >120 sec or DNF → 0 points

GRADE MULTIPLIER:
  Grade 5:  2.0x (fastest earning, youngest)
  Grade 6:  1.85x
  Grade 7:  1.70x
  Grade 8:  1.55x
  Grade 9:  1.40x
  Grade 10: 1.25x
  Grade 11: 1.10x
  Grade 12: 1.0x (baseline, oldest)

SCHOOL DIVISION:
  Division A: 8+ students
  Division B: 4-7 students
  Division C: 0-3 students

BONUSES (all multiplied by grade multiplier):
  PB Bonus: +1 pt (beat personal record)
  Clutch: +2 pts (PB in finals)
  Streak: +3 pts (3+ consecutive improvements)
  School Momentum: +5 pts per student (school had 0 DNFs)
